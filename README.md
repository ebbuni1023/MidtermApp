# MidtermApp

Summer 2021 React-native Lets talk Money Midterm App Project

## Table of contents
* [Setup](#technologies)

## Technologies
Project is created with:
* React-native
* Expo
* Firebase
* Google admob
	
## Setup
To run this project, install it locally using expo:

```
$ cd ./MidtermApp
$ yarn install or npm install
$ expo start
and press a to android, i to ios
```
